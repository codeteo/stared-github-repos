#  Most starred Github repos (Coding Challenge)

## Introduction

> A small android app that lists the most starred Github repos that were created in the last 30 days.

>it uses the Model View Presenter (MVP) pattern.
libraries used :
* Retrofit + RxJava
* ButterKnife
* Picasso
